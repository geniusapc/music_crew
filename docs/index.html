Project Setup

  1. This project assumes you have NodeJS v10 or greater installed.

  2. You should also have npm v6 or greater installed as well (this comes packaged with Node 8).

  3. You'll also need a recent version of git installed as well.

  4. You have to install mongoDb so that you can access database.

  5. You have to install redis

Getting Started
  1. Create a .env on the project root directory and type the following gmailUser= gmailPass= REDIS_URL= MONGODB_URI=

2) Start the server locally by running npm run dev making sure you are on the project root directory; the server will then start on http://localhost:8000/graphql
